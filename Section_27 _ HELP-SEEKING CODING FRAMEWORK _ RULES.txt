OUTPUT SECTION 27 — HELP-SEEKING CODING FRAMEWORK & RULES
──────────────────────────────────────────
This section explicitly defines how student messages to the teacher/system are categorized and translated into adaptive responses.

### 27.1 Help-Seeking Coding Framework
| Code | Category | Description | Example | Adaptive Interpretation |
|------|----------|-------------|---------|-------------------------|
| **HS1** | Feedback clarification | Student asks about teacher/system feedback | "I do not understand the cohesion comment." | Positive sign of feedback literacy; triggers `dialogic_scaffolding`. |
| **HS2** | Linguistic assistance | Student asks about grammar, vocabulary, or form | "Is this verb tense correct?" | Language-focused; triggers `metalinguistic_feedback` or `lexical_refinement`. |
| **HS3** | Argument development | Student asks about thesis, evidence, reasoning | "Should I add another example here?" | High-value academic help-seeking; triggers `argument_expansion`. |
| **HS4** | Procedural clarification | Student asks about requirements, deadline, length | "How many sources should I include?" | Minimal help-seeking; useful but less cognitively deep. |
| **HS0** | No help-seeking | No message sent | N/A | Risk ONLY when combined with weak performance. |

### 27.2 Help-Seeking Thresholds
| Variable | Low / Risk | Moderate / Watch | Strong / Adaptive |
|----------|------------|------------------|-------------------|
| **help_message_count** | 0 when performance is weak | 1 procedural message | >= 1 clarification, argument, or language message |
| **help_message_quality**| None / irrelevant | Procedural only | Feedback clarification, language assistance, or argument dev |
| **help_seeking_timing** | No message before/after feedback | Message before deadline only | Message after feedback and before revision |
| **help_seeking_status** | hidden_difficulty_risk | minimal_help_seeking | adaptive_help_seeking |

### 27.3 Help-Seeking Rules
| Condition | Learner State | Rule ID | Feedback Template | Teacher Action |
|-----------|---------------|---------|-------------------|----------------|
| Weak criterion score <= 2 + no help msg | `help_seeking_risk = High` | D5 | `metacognitive_prompt` | Proactive teacher intervention |
| At least one feedback-clarification message | `help_seeking_regulation = Adaptive`| D4 | `dialogic_scaffolding` | Short clarification conference |
| At least one argument-development message | `argument_help_seeking = Adaptive` | B2/F1 | `argument_expansion` | Guide reasoning and evidence integration |
| At least one language-assistance message + weak score | `linguistic_help_needed` | B7/B9 | `metalinguistic_feedback` | Target grammar or vocabulary |
| Only procedural message + weak writing performance | `minimal_help_seeking` | A2/D3 | `planning_prompt` | Move learner from procedural to strategic regulation |
