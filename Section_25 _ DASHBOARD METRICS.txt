OUTPUT SECTION 25 — DASHBOARD METRICS
──────────────────────────────────────────
The system will provide a dashboard for the teacher/researcher to monitor the overall health of the class. The dashboard aggregates the following counts directly from the backend database:

| Metric | Formula / Count Logic |
|--------|-----------------------|
| Total students | 60 (50 logged) |
| Completed teacher decisions | Count of rows where Teacher_Decision != Pending |
| High argument risk | Count of rows where Argument_Risk = High |
| High cohesion risk | Count of rows where Cohesion_Risk = High |
| High language risk | Count of rows where Language_Risk = High |
| High forethought risk | Count of rows where Forethought_Risk = High |
| High revision risk | Count of rows where Revision_Risk = High |
| High feedback uptake risk | Count of rows where Feedback_Uptake_Risk = High |
| High help-seeking risk | Count of rows where Help_Seeking_Risk = High |
| Strategic writers | Count of Learner_Profile = "Strategic" |
| Engaged but developing writers | Count of Learner_Profile = "Engaged but developing" |
| Efficient but passive writers | Count of Learner_Profile = "Efficient but passive" |
| At-risk writers | Count of Learner_Profile = "At-risk writer" |
